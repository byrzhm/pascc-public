## 团队成员基本素质

1. 具有基本的编程能力，包含但不限于算法与数据结构，C++面向对象编程与Vistor设计模式
2. 能够清楚地认识自己被分配的任务，并主动进行相关的培训，包含但不限于flex，bison工具的学习
3. 熟悉git版本管理工具，能够高效地进行多人合作编程

## 基本要求

1. 以测试为导向，永远不提交没有测试过的代码
2. 要求对每一个类与方法提供注释
3. 自觉完成每周的学习与编程任务
4. 诚信，如果借鉴了开源代码，则必须在代码和文档中提供链接
